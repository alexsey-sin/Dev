Часто задаваемые вопросы
--------------------------

В: С каким объемом данных может справиться Flot?

A: Flot с радостью примет все, что вы отправите ему, так что ответ
зависит от браузера. Эмуляция excanvas, используемая для IE (созданная с
VML) делает IE самым медленным браузером, поэтому обязательно протестируйте его.
если пользователи IE входят в вашу целевую группу.

1000 баллов - это не проблема, но как только вы начнете получать больше
точек, чем ширина пикселя, вы, вероятно, должны начать думать о
субдискретизация/агрегация, так как это близко к пределу разрешения
график в любом случае. Если вы понижаете частоту дискретизации на стороне сервера, вы также экономите пропускную способность.


В: Flot не работает, когда я использую данные JSON в качестве источника!

A: На самом деле, Flot любит данные JSON, вы просто ошиблись форматом.
Дважды проверьте, что вы не вводите строки вместо чисел,
например [["0", "-2.13"], ["5", "4.3"]]. Это самая распространенная ошибка, и
ошибка может появиться не сразу, потому что Javascript может сделать некоторые
преобразование автоматически.


В: Могу ли я экспортировать график?

О: Это ограничение технологии холста. есть крючок
объект холста для вывода изображения, но вы не получите галочку
этикетки. И вряд ли он будет поддерживаться IE. В этот момент ваш
лучше всего, вероятно, сделать снимок экрана, например. с помощью PrtScn.


В: В режиме времени все полосы крошечные?

A: На самом деле невозможно определить ширину полосы автоматически.
Таким образом, вы должны установить ширину с параметром barWidth, которого нет в
пикселей, но в единицах оси x (или оси y для горизонтального
бары). Для режима времени это миллисекунды, поэтому значение по умолчанию равно 1.
делает полосы шириной 1 миллисекунду.


В: Могу ли я использовать Flot с такими библиотеками, как Mootools или Prototype?

О: Да, Flot поддерживает его из коробки, и это просто! Просто используйте jQuery
вместо $, например. вызовите jQuery.plot вместо $.plot и используйте
jQuery(что-то) вместо $(что-то). Для удобства можно
поместите элемент DOM для заполнителя графика, где примеры и
документация API использует объекты jQuery.

В зависимости от того, как вы включаете jQuery, вам, возможно, придется добавить одну строку
код для предотвращения перезаписи jQuery функций из других
библиотеки, см. документацию по jQuery ("Использование jQuery с другими
библиотеки") для получения подробной информации.


Q: Flot не работает с [widget framework xyz]!

A: Проблема, скорее всего, внутри фреймворка или использования вами
рамки.

Единственная нестандартная вещь, используемая Flot, — это тег canvas; в противном случае
это просто серия div с абсолютным позиционированием внутри
тег-заполнитель, который вы вставили. Если это не работает, возможно, это
потому что используемый вами фреймворк делает что-то странное с
ДОМ. В крайнем случае, вы можете попробовать перестроить и посмотреть, поможет ли это.

Если вы обнаружите, что мы можем помочь Флоту чем-то конкретным, почувствуйте себя
бесплатно отправить отчет об ошибке. В противном случае вы можете обратиться за помощью
в списке рассылки, но, пожалуйста, не отправляйте отчет об ошибке во Flot -
вместо этого попробуйте фреймворк.
