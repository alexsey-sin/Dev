Множества

Вы узнали, как работают словари и уже можете написать код, который напечатает
все любимые группы пользователя. Но всегда ли этот код будет работать хорошо?

Попробуйте запустить программу:

	favorite_songs = {'У шамана три руки': 'Пикник',
					  'Звезда по имени Солнце': 'Кино',
					  'Восьмиклассница': 'Кино',
					  'Тополиный пух': 'Иванушки International',
					  'Группа крови': 'Кино'}

	print(', '.join(favorite_songs.values())) 

Как видите, в случае, если у нескольких песен один и тот же исполнитель,
он появляется в выводе несколько раз. А зачем эти повторения? Вот когда нам
поможет новый тип данных set. Множества часто называют сетами.

Тип set очень похож на список, но есть два важных отличия:

1) Элементы в сете не повторяются, он избегает повторов, автоматически
не вписывая их;

2) Не гарантируется, что при выводе элементов на экран будет соблюден какой-то
определенный порядок.

Любой список можно превратить в сет простым вызовом функции set() :

	bands = ['Пикник', 'Ария', 'Блестящие', 'Блестящие']
	# получаем сет unique_band_names (с англ. «уникальные названия групп»)
	unique_band_names = set(bands) 

По элементам сета тоже можно пройти циклом, например вот так:

	for band in unique_band_names:
		print('Доктор, я не могу больше слушать группу ' + band)

Задача 1/2:

Сделайте из списка cities сет unique_cities, где записаны по разу названия
городов, в которых живут ваши друзья. После этого напечатайте строку из
элементов unique_cities на экран через запятую и пробел — да, join() работает
и для множеств тоже!

Подсказка:

Чтобы сделать из списка множество, вызовите функцию set().

******************************************
	cities = ['Вологда', 'Чебоксары', 'Тольятти', 'Вологда']

	unique_cities = set(cities) # напишите здесь свой код
	print(', '.join(unique_cities))
******************************************

Задача 2/2:

Для каждого города в списке cities напечатайте на экран сообщение У меня есть
друг в городе <название города>. Сообщения не должны повторяться: фраза про
каждый город должна быть выведена лишь один раз.

Подсказка:

Сделайте из списка сет и пройдите по нему в цикле for.

******************************************
	cities = ['Санкт-Петербург', 'Хабаровск', 'Казань', 'Санкт-Петербург', 'Казань']

	unique_cities = set(cities)
	for city in unique_cities:
		print('У меня есть друг в городе', city)
******************************************
